# PDF RAG Chatbot

An AI-based chatbot that answers questions from PDF documents using Retrieval-Augmented Generation (RAG).

## Overview

This project enables users to upload PDF files and query their content. The system retrieves relevant document chunks and generates accurate, context-aware responses using a language model.

## Features

- PDF-based question answering
- Context-aware response generation
- Multilingual support (Hindi and English)
- Text-to-speech output
- Efficient document retrieval using embeddings

## Tech Stack

- Python
- LangChain
- Mistral AI
- HuggingFace Embeddings
- ChromaDB
- Streamlit
